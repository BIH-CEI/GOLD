This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide

Profile Body Height

Profile for body height of a patient
Profile Body Weight

Profile for body weight of a patient
Profile ECOG Performance Status

Profile for the Eastern Cooperative Oncology Group (ECOG) Performance Status that represents a cancer patient’s functional status.
Profile Histology Topography

Profile for the classification of a tumor’s morphology(histology) and topography using ICD-O-3.
Profile Histopathological Tumor Grading WHO

Profile for the description of a tumor’s grade using the general four-tier grading scheme following the recommendations of the WHO classification of tumors and UICC.
Profile Karnofsky Performance Status

Profile for the Karnofsky Performance Status to measure a cancer patient’s ability to perform ordinary tasks.
Profile Vital Signs Base

Profile for the base definition of all vital signs. May not be used directly. Only for reprofiling purposes.

Terminology: Value Sets

These define sets of codes used by systems conforming to this implementation guide

ValueSet ECOG Performance Status SNOMED

The ValueSet contains the ECOG Performance Status values from SNOMED CT.
ValueSet Histologic Grade SNOMED

The ValueSet contains codes for the tumor grading system following the recommendations of the WHO Classification of Tumours from SNOMED CT.
ValueSet Morphology ICD-O-3

The ValueSet contains the morphology (histology) codes from ICD-O-3.
ValueSet Morphology SNOMED

The ValueSet contains the morphology (histology) codes from SNOMED CT.
ValueSet Topography ICD-O-3

The ValueSet contains the topography codes from ICD-O-3.
ValueSet Topography SNOMED

The ValueSet contains the topography codes from SNOMED CT.
